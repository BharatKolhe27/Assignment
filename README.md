#  React Redux Product Dashboard

A simple React application demonstrating **state management using Redux Toolkit**, reusable components, custom hooks, and basic cart functionality.

---

##  Features

- Reusable `<PostCard />` component
-  Language Switcher (English ↔ French) using global state
-  Product Dashboard with API integration
-  Cart functionality (Add, Remove, Update Quantity)
- Global state management using Redux Toolkit
- Custom Hook (`useFetch`) for API handling

## Concepts Covered

- React Hooks (`useState`, `useEffect`)
- Custom Hooks (`useFetch`)
- Redux Toolkit (`createSlice`, `configureStore`)
- `useSelector` & `useDispatch`
- Component Reusability
- API Fetching
- State Management
- Conditional Rendering

---


## How It Works

### Product Dashboard
- Fetches products from API  
- Displays top 5 products  
- Allows adding products to cart  

### Cart System
- Add product to cart  
- Remove product  
- Update product quantity  

### Language Toggle
- Switch UI text between English & French using Redux  

### Custom Hook (`useFetch`)
- Handles API calls and loading state
